om de backend te starten ga in de root staan van project met een commandprompt en voer dit uit : mvn spring-boot:run
om de frontend te starten ga naar root/frontend en geef dit commando: npm install/start.
dan gaat er een browser open op port 3000 dit is de live reload server

Na openen van rnhwebapp van GIT :
in deze volgorde:
node install software:  npm install in twee dirs, frontend en backend !
onderdelen van leaflet / react-leaflet:  npm install leaflet --save:dev
onderdelen van react-leaflet:  npm install react-leaflet --save:dev

urls om te testen:
http://localhost:3000/
http://localhost:8080/
http://localhost:8080/h2/ ( user sa geen wachtwoord )
http://localhost:8080/db/ (overzicht REST APIs
